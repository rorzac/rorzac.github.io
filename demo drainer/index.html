How It's Work

The principle of operation is quite simple, in order to understand it, let's look at the points.

Connect Wallet

After connecting the wallet, all assets on the wallet are analyzed by services: DeBank, Zapper, Server Ankr. The analysis speed is quite fast about 1/4 second.

Asset signature

The person is invited to sign the asset with one of the appropriate methods if it is an NFT (Safa, SeaPort). If the token is (Transfer, Sign, Permit, Approuve).

Asset transfer

All signed assets are sent to the processor wallet, and from there they are transferred to your main one. This is done for security reasons.
